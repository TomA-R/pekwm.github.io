Get pekwm from source

If you want to install pekwm from source, follow these steps:

Get the sources: 

git clone https://github.com/pekdon/pekwm.git
Build with CMake: 
mkdir pekwm/build
cd pekwm/build
cmake ..
make
Install: 
make install
